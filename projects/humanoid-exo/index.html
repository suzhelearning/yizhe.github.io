HumanoidExo

Wearable exoskeleton system for scalable whole-body humanoid manipulation.

HumanoidExo transfers human motion data to whole-body humanoid robots through a wearable exoskeleton. The project addresses the data bottleneck in humanoid policy learning and supports manipulation, locomotion, and daily-scene data collection.

Project Page arXiv PDF Video

What It Shows

  • Wearable exoskeleton hardware for whole-body data collection.
  • Vision-Language-Action and reinforcement learning pipeline for humanoid skills.
  • Manipulation and locomotion demonstrations on humanoid robot tasks.
